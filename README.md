# lrfasd.github.io
>以下工具基于html5+javascript开发，具有如下特点：
>1. 支持Windows、Mac、Linux、iOS、Android等系统；
>2. 使用时仅需支持html5的浏览器，绿色、轻量、便携；
>3. 代码开源，避免后门，保证使用安全；
>4. 可纯客户端操作，无需任何流量。
